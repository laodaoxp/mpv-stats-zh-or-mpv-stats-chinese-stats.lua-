## 📖 项目简介

这是一个 **mpv播放器统计信息脚本** 的中文翻译版。  
原版由 [Argon-](https://github.com/Argon-) 开发，本版本将所有界面文字翻译为中文，方便中文用户使用。

> 🎨 **界面清爽 | 功能完整 | 即装即用**

---

## ✨ 功能特点

| 功能 | 说明 |
|:---|:---|
| 🌐 **全中文界面** | 所有显示文字、提示信息都翻译为中文 |
| 🚀 **功能完整** | 保留原版所有功能，无任何删减 |
| 🎯 **即装即用** | 复制到 scripts 文件夹即可使用 |
| 🔧 **兼容性好** | 可与任何 mpv 综合配置共存 |
